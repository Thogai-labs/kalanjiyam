Code and Data

All resources, datasets, and infrastructure powering the platform.

{# MAIN CARD #}

Codebase

All of our code is available on GitHub. We use Python (Flask) for the backend, Tailwind for CSS, and native JavaScript for the rest. Our site is hosted on DigitalOcean.

Text Sources

Much of our text data comes from GRETIL. For the Ramayana and Mahabharata, we use official electronic editions from John D. Smith's archive.

As the project grows, we will integrate more datasets and eventually generate new texts.

Dictionary Sources

  • Most dictionary data comes from Cologne Digital Sanskrit Dictionaries .
  • Sanskrit-Hindi dictionary is from Samsaadhanii project .
  • Additional dictionary data comes from indic-dict .

Linguistic Data

Parse data comes from the Digital Corpus of Sanskrit (DCS) project.
