{{ _h('what-is-our-goal', 'What is our goal?') }}

We want to create machine-readable texts that we can display on Kalanjiyam and share with others — at the highest quality we are capable of.

{{ _h('the-golden-rule', 'The golden rule') }}

Don't change what the page says.

Follow the book's spelling, script, and word-splitting conventions exactly. Include everything meaningful on the page.

{{ _h('characters', 'Characters') }} {% call components.div_note() %} Rules that start with [auto] can be fixed with software — not a good use of time. {% endcall %} {{ _h2('Hyphens at the end of a line') }}

If the printed line ends with a hyphen, keep it. Our program uses these hyphens to stitch together different lines.

{{ _h2('Hyphens and dashes') }}
  • Hyphens (-) — write with a single dash -
  • Em dashes (—) — write with two dashes --
{{ _h2('[auto] Spaces around punctuation') }} {{ _compare('arthaH -- eka', 'arthaH--eka') }} {{ _h2('[auto] Quotation marks') }} {{ _compare('"naraH"', '"naraH"') }} {{ _h('paragraphs', 'Paragraphs') }} {{ _h2('Line breaks and hyphens') }}

Keep line breaks and line-ending hyphens. They make proofreading easier by keeping the digitized text similar to the image.

{{ _h2('Spaces between paragraphs') }}

Separate paragraphs and verses with blank lines.

{{ _h2('Page headers and footers') }}

Delete page headers and footers — they don't have useful information.

{{ _h2('Footnotes') }}

Mark footnote references with brackets and a caret:

{{ _compare('naraH1', 'naraH[^1]') }} {{ _h('pages', 'Pages') }}

Skip any content not part of the original book: handwritten notes, stamps, watermarks, dirt, stains, etc.
